# Unit Testing

## Prerequisites

Install `g++`, `CppUnit`, and `gcovr`.

## Execute the Tests

1. Compile
   ```{bash}
   make -f TestMakefile
   ```
2. Run the tests
   ```{bash}
   ./test.out
   ```
3. Show coverage report
   ```{bash}
   <!-- TODO: Cover all cpp files in the parent directory when generating the coverage report -->
   gcovr
   ```
